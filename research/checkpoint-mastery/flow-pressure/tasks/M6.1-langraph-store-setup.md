## Code Pattern

```python
from langgraph.store.postgres import PostgresStore

# Create store (shares PostgreSQL with checkpoints)
store = PostgresStore(
    conn_string="postgresql://user:password@host:5432/checkpoints"
)
await store.setup()

# Store operations
# Put
await store.put(
    namespace=["user_preferences", "user-123"],
    key="theme",
    value={"color": "dark", "font": "large"}
)

# Get
pref = await store.get(
    namespace=["user_preferences", "user-123"],
    key="theme"
)

# Search
results = await store.search(
    namespace=["learned_facts"],
    query="Paris is the capital"
)
```

## Notes

**Store vs Checkpoints:** Checkpoints = thread state (temporal). Store = persistent facts (timeless).

**Namespace Strategy:** Organize by data type (preferences, facts, knowledge). Enables efficient queries.
